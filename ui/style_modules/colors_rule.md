# Правила для цветов в SkillBot

## Формат цветов

Все цвета в приложении должны использовать корректный шестнадцатеричный формат:

- 3-значный формат: `#RGB` (например, `#444`)
- 6-значный формат: `#RRGGBB` (например, `#44444`)

## Недопустимые форматы

Следующие форматы НЕ допускаются:
- `#4444` (4 символа)
- `#44444` (5 символов)
- `#4444444` (7 символов)

## Использование переменных

Предпочтительно использовать переменные из `colors.py`:
- `PRIMARY_COLOR = "#0078d4"`
- `SECONDARY_COLOR = "#ff0066"`
- `BACKGROUND_COLOR = "#1e1e1e"`
- `PANEL_COLOR = "#2d2d2d"`
- `HIGHLIGHT_COLOR = "#00d4ff"`
- `SUCCESS_COLOR = "#4caf50"`
- `WARNING_COLOR = "#ffc107"`
- `DANGER_COLOR = "#f44336"`
- `TEXT_COLOR = "#e0e0e0"`
- `TEXT_SECONDARY = "#b0b0b0"`
- `TEXT_DISABLED = "#6666"`
